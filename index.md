## Features

- **Book Cover Generation** - Automated book cover creation
- **Social Media Assets** - Generate banners, GIFs, and promotional materials
- **Effects Tools** - Title inversion, cropping, and banner controls
- **Mint & Print** - Print-ready output generation
- **Dockable Panel** - Dock alongside native After Effects panels
- **JSON Configuration** - Dynamic settings via ae_bridge.json

## Quick Start

### Dockable Panel Installation (Recommended)
1. Copy folder to ScriptUI Panels folder
2. Restart After Effects
3. Launch from Window menu

### Floating Palette Installation
1. Copy folder to Scripts folder
2. Launch from File > Scripts
